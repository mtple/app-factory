# Snap Handoffs

App Factory is separate from Snap Factory. When a Snap project needs a Farcaster Mini App surface, hand off only the public product requirements:

- App name, subtitle, and description.
- Primary category.
- Public base URL and desired home path.
- Icon, splash, and embed artwork.
- Webhook behavior and event expectations.
- Farcaster account association values.

Do not copy Snap secrets or wallet-specific implementation details into this repo.
